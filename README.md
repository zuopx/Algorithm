# Classic Algorithm Problems

## Knapsack

### 0/1 Knapsack Problem

maximize 
$$\sum_{i=1}^n w_i x_i$$
subject to
$$\sum_{i=1}^n w_i x_i \le c, \ and \ \  x_i \in \{0, 1\}, 1 \le i \le n.$$
